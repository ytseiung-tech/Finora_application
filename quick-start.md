## 故障排除

### 如果遇到错误：
1. 确保在 `Expo` 目录中运行命令
2. 删除 `node_modules` 并重新安装：
   ```bash
   rm -rf node_modules
   npm install
   ```
3. 清除 Expo 缓存：
   ```bash
   npx expo start --clear
   ```

### 常见问题：
- **端口被占用**: 选择使用其他端口 (Y)
- **依赖版本不匹配**: 运行 `npm install` 更新
- **组件未找到**: 确保所有导入路径正确
