# 需要添加的圖片

請在以下路徑添加這三張圖片：

## 1. `assets/customize/customize.png`
- 用途：自訂主題按鈕圖示
- 建議尺寸：40x40 pixels
- 建議內容：調色盤或畫筆圖標

## 2. `assets/customize/save.png`
- 用途：我的主題按鈕圖示
- 建議尺寸：40x40 pixels
- 建議內容：儲存圖標或資料夾圖標

## 3. `assets/customize/upload.png`
- 用途：提交的主題按鈕圖示
- 建議尺寸：40x40 pixels
- 建議內容：上傳或發送圖標

添加這三張圖片後，應用程式即可正常運行。
